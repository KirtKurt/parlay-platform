# Silvers Syndicate UX Website Remap

## Goal

Make the site easier for a new visitor to understand in the first 10 seconds, while still pushing the strongest conversion path: free-week registration.

## Primary visitor types

### 1. The user who already has a pick

This user is closest to buying. They are already emotionally committed to a side or a parlay. The best page for them is the negative-message page.

Primary path:

1. Home
2. Test Picks
3. Picks Audit
4. Register for free week
5. Unlock full market board

### 2. The user browsing by sport

This user wants to know whether the platform supports the sport they care about.

Primary path:

1. Home
2. Sports
3. Sport-specific board
4. Pricing
5. Register

### 3. The skeptical user

This user needs to understand the logic before paying.

Primary path:

1. Home
2. Start Here
3. Methodology
4. Picks Audit
5. Register

### 4. The returning user

This user needs less education and more direct access.

Primary path:

1. Login
2. Account
3. Sports board or admin path depending on role

## Site navigation after remap

Top navigation:

- Start Here
- Test Picks
- Sports
- Pricing
- Login
- Start Free Week

This reduces clutter and puts the highest-intent path near the front.

## Page strategy

### Home

The home page should not explain every feature. It should answer:

- What is this?
- Why should I care before building a ticket?
- Where should I go next?

The home page now presents three clear paths:

- I already have a pick
- I want to browse today's board
- I want to know how this works

### Start Here

The Start Here page acts like onboarding before account creation. It tells users what to do first and why.

### Picks Audit

The Picks Audit page is the negative-message landing page. It creates urgency by challenging the user's confidence.

### Sports

The Sports page exists for organic search and sport-specific entry. Each sport page should eventually target terms like:

- NFL parlay risk checker
- College football line movement tracker
- NBA market movement analysis
- NHL parlay risk intelligence

### Pricing

Pricing should stay simple: Core and Pro only. No Starter. No Enterprise.

### Register

The register page should reinforce the first-week-free offer and explain that premium access unlocks the full board.

## Conversion principle

Do not ask the visitor to understand everything. Ask them to test one pick.

Best CTA language:

- Test Your Picks
- Start Free Week
- Preview the Board
- Compare Core vs Pro

Avoid language that sounds too technical or too guaranteed.

## Compliance note

The site must keep clear positioning:

- Sports market intelligence
- Informational and entertainment use
- No guaranteed outcomes
- No sportsbook operation
- No financial promise

# SEO, Legal, and Compliance Baseline

This document tracks the baseline protections added to the Silvers Syndicate frontend.

## Added

- Global SEO metadata
- Open Graph metadata
- Twitter card metadata
- Organization structured data
- Public sitemap
- Public robots file
- Global footer links
- Privacy page
- Site rules page
- Disclaimer page
- Safe use page
- Accessibility page
- Contact page

## Product positioning

Silvers Syndicate is positioned as a sports market intelligence and education platform. The UI should avoid promising wins, locks, guaranteed outcomes, or certain predictions.

## Required next review before launch

Before accepting live subscribers, counsel should review:

- Privacy policy
- Site rules
- Disclaimer
- Subscription cancellation/refund language
- Age and jurisdiction controls
- Payment provider terms
- Data retention policy
- Accessibility statement
- Cookie consent requirements

## Operating principle

Do not store payment credentials in the frontend. Do not hard-code secrets. Keep subscription confirmation server-side.
